Navigating and Viewing The Internet

Understanding that the internet is really a web of computers interacting with each other thru commands and computer language. Anytime a person wants to "go somewhere" on the internet they have to type in an address on their browser like www. udacity.com and the computer sends a HTTP Request to a server. Udacity is where you end up. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.


HTML

HTML stands for Hypertext Markup Language. HTML documents make up most of what we see on the internet. It is written in the form of HTML elements consisting of tags enclosed in angle brackets. Here is an example


Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. < br > is an empty element without a closing tag.


Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Some examples of inline elements are : a, span, br, img, and strong. Some box elements are: p, div, and form.
